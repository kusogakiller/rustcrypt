# RustCrypt

Safe. Fast. Cryptographic.

---

## Overview

RustCrypt is a lightweight cryptographic utility for generating secure, entropy-based keys and passwords.

Built with Rust for performance, safety, and simplicity.

---

## Features

- High-entropy key generation
- Deterministic cryptographic pipeline
- Minimal CLI interface
- No external dependencies

---

## Philosophy

Security through simplicity.  
Minimal input, maximal entropy.
